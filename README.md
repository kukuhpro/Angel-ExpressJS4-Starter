## Angel 
Angel is not a framework, is a ExpressJS 4  Starter Kit.
This Starter is version 2.0.0

### Installation
##### 1. running npm install 
```
$ npm install
```
##### 2. copy Configuration file
```
$ cp .env.example .env
```
##### 3. run on your app by running 
```
npm run start
```
##### or
```
node app.js
```
