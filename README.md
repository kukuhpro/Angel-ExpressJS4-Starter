## Angel 
Angel is not a framework, is a Express 4 Bootstrap Starter. 

### Installation
##### 1. running npm install 
```
$ npm install
```
##### 2. copy Configuration file
   ```
   $ cp .env.json.example .env.json
   ```
##### 3. run on your app by running 
```
npm start
```
##### or
```
node app.js
```
